## 🎨 Scoring Algorithm

RepoRadar uses a smart 100-point scoring system:

```
Score = Name(40) + Description(30) + Topics(20) + Popularity(10)
```

**Example:**
- Query: "open directory indexer"
- Repo: "h5ai" (file indexer)
- Score: 85% (high match!)
